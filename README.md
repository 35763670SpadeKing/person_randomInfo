# person_randomInfo

批量生成随机个人信息并导出到 Excel 的 Python 工具。

## 功能特性
- 生成大量随机中国个人信息（姓名、性别、出生日期、手机号、地址、邮箱等）。
- 支持导出为 Excel 文件。
- 可自定义生成数量和字段。

## 安装依赖
```bash
pip install openpyxl
```

## 使用方法
```bash
git clone https://github.com/35763670SpadeKing/person_randomInfo.git
cd person_randomInfo
python 主脚本.py
```
